# КАМЕНЪ — сайт столешниц из искусственного камня (Иркутск)

Статический сайт для сбора заявок на столешницы из искусственного камня.
Заявки уходят тебе **на e-mail** (и при желании в Telegram), а ты пересылаешь их заводам-изготовителям.

---

## 📁 Что внутри

```
сайт столешницы/
├── index.html                      ← главная (лендинг)
├── materialy/
│   ├── kvarcevyj-aglomerat.html    ← SEO-страница: кварц
│   ├── akrilovyj-kamen.html        ← SEO-страница: акрил
│   └── litevoj-mramor.html         ← SEO-страница: мрамор
├── assets/
│   ├── css/style.css               ← весь дизайн
│   ├── js/main.js                  ← формы, калькулятор, галерея
│   └── img/                        ← сюда положить фото работ
├── robots.txt
├── sitemap.xml
└── README.md                       ← этот файл
```

Дизайн: **Dark Luxury** (чёрно-золотая палитра, шрифты Cormorant Garamond + Inter).

Используемые библиотеки (open-source с GitHub, подключены через CDN):
- **Lenis** (darkroom.engineering) — плавный инерционный скролл, как на сайтах-победителях Awwwards
- **GSAP + ScrollTrigger** — проявление текста/блоков, параллакс фото, счётчики
- **Swiper** — слайдер галереи работ
- **GLightbox** — увеличение фото по клику

> Фото сейчас подгружаются с Unsplash (бесплатные) с двойным фолбэком. Замени их на свои реальные
> фото объектов: положи в `assets/img/gallery/` и поменяй `src` у тегов `<img>` (см. раздел «Добавить фото»).

## 🚀 ЗАПУСК ЗА 4 ШАГА

### Шаг 1. Подключить приём заявок на почту (5 минут, бесплатно)

1. Зайди на **https://web3forms.com** → введи свой e-mail (например `andreimg500@gmail.com`) → получишь **Access Key** (строку вида `a1b2c3d4-...`).
2. Открой файл `assets/js/main.js`, найди строку:
   ```js
   const WEB3FORMS_KEY = 'ВАШ_ACCESS_KEY';
   ```
   и вставь свой ключ между кавычками.
3. Всё — теперь каждая заявка с любой формы сайта придёт тебе на почту.

> Web3Forms бесплатен до 250 заявок/мес. Заявки приходят с темой
> «Новая заявка с сайта столешниц (Иркутск)» и содержат имя, телефон, материал и комментарий.

**Дублировать заявки в Telegram** (по желанию): в личном кабинете Web3Forms
включи интеграцию Telegram/webhook — настраивается мышкой, код менять не нужно.

### Шаг 2. Вписать свои контакты

Замени заглушки во **всех** `.html` файлах (главная + 3 страницы материалов):

| Заглушка | На что заменить |
|---|---|
| `+7 (XXX) XXX-XX-XX` и `+7XXXXXXXXXX` | твой реальный телефон |
| `info@lithos-irk.ru` | твой e-mail |
| `Иркутск, ул. ______, д. __` | адрес (или убери, если работаешь без офиса) |
| `lithos-irk.ru` | твой реальный домен (после покупки) |

Быстрый способ — открыть папку в редакторе (VS Code / Notepad++) и сделать
«Найти и заменить по всем файлам» (Ctrl+Shift+H в VS Code).

### Шаг 3. Купить домен и хостинг

**Рекомендуемые домены** (ключ + регион, проверь свободность на reg.ru / nic.ru):

1. **lithos-irk.ru** ← основной вариант (короткий, есть город и ключ)
2. stoleshnitsy-irk.ru
3. kamen-irkutsk.ru
4. kvarc-irkutsk.ru
5. stoleshnica-irk38.ru *(38 — код региона)*

Домен `.ru` стоит ~200–300 ₽/год.

**Хостинг** (любой подойдёт, сайт статический и лёгкий):
- **Beget, Reg.ru, Timeweb** — обычный хостинг, ~150–300 ₽/мес. Просто загрузи файлы в папку сайта по FTP.
- **Бесплатно**: GitHub Pages, Netlify, Cloudflare Pages — но для `.ru` SEO лучше российский хостинг.

**Как залить:** загрузи всё содержимое папки (index.html, materialy, assets, robots.txt, sitemap.xml) в корень сайта на хостинге.

### Шаг 4. Подключить SEO-инструменты

1. **Яндекс.Вебмастер** (https://webmaster.yandex.ru) — добавь сайт, подтверди права, отправь `sitemap.xml`. Главный источник трафика в РФ.
2. **Яндекс.Метрика** + **Google Search Console** — статистика и цели.
3. Зарегистрируй компанию в **Яндекс.Бизнес (Карты)** и **2ГИС** — для Иркутска это даёт «тёплые» заявки.

После установки Метрики впиши её номер в `index.html` (счётчик) — в коде уже
заложена отправка цели `lead` при отправке формы (`window.ym(...)`).

## 🧱 Страницы изделий (SEO) и каталог

В папке `uslugi/` лежат **25 SEO-страниц** под отдельные изделия (кухонные столешницы,
подоконники, барные стойки, мойки, душевые поддоны, ресепшн, фартуки и т.д.) + страница-каталог
`uslugi/index.html`. Каждая страница — со своим title, description, текстом, микроразметкой и формой.

Эти страницы **генерируются** из файлов данных, чтобы их было легко править массово:
- `uslugi/_data.json` — контент (заголовки, тексты, преимущества) по каждой странице;
- `uslugi/_faq.json` — мини-FAQ (вопросы-ответы) по каждой странице;
- `uslugi/_images.json` — какое фото (id с Pexels) ставить на странице;
- `uslugi/_build.ps1` — генератор: собирает все страницы + каталог + обновляет `sitemap.xml`.

> Фото берутся с **Pexels** (бесплатная лицензия) по id из `_images.json`. Чтобы поставить
> своё фото — замени значение в `_images.json` на нужный id с pexels.com (или впиши прямой URL
> в шаблон) и пересобери. На главной фото можно менять прямо в `index.html`.

**Чтобы изменить текст или добавить новое изделие:**
1. Откройте `uslugi/_data.json`, поправьте нужный блок или скопируйте один объект `{...}` и
   измените `slug`, `title`, `h1`, тексты и т.д.
2. Запустите генератор: в PowerShell в папке сайта выполните
   `pwsh ./uslugi/_build.ps1` (или правый клик по `_build.ps1` → «Выполнить с помощью PowerShell»).
3. Страницы и `sitemap.xml` пересоберутся автоматически.

> Файлы `_data.json` и `_build.ps1` — служебные (для сборки), на хостинг их можно не загружать.

## 🖼 Добавить фото работ

Сейчас галерея и карточки используют «каменные» CSS-текстуры-заглушки.
Чтобы заменить на реальные фото:

1. Положи фото в `assets/img/gallery/` (например `kuhnya-1.jpg`).
2. В `index.html` в блоке `<!-- ГАЛЕРЕЯ РАБОТ -->` замени
   `<div class="stone stone--white"></div>` на
   `<img src="assets/img/gallery/kuhnya-1.jpg" alt="Кухонная столешница из кварца, Иркутск">`.
3. Для увеличения по клику добавь ссылке `href` путь к фото:
   `<a class="gallery-slide glightbox" href="assets/img/gallery/kuhnya-1.jpg">`.

> `alt` у фото пиши с ключевыми словами и городом — это плюс к SEO.

## 📈 Что улучшать дальше (SEO-рост)

- Добавить страницы под запросы: «столешница для кухни», «подоконники из камня»,
  «столешница в ванную», «барная стойка из камня» — структура уже готова (папка `uslugi/`).
- Завести блог: «акрил или кварц», «как ухаживать за столешницей» — это собирает
  информационный трафик и ведёт на заявки.
- Реальные фото и отзывы (со скринами из 2ГИС/Яндекс) повышают конверсию.
- Цены и УТП — держи актуальными в `index.html` и `main.js` (калькулятор).

Цены в калькуляторе и на страницах сейчас ориентировочные — поставь свои.
Калькулятор: `assets/js/main.js`, объект `PRICE`.
```js
const PRICE = { acryl: 16000, quartz: 24000, marble: 14000 }; // ₽ за пог. метр
```
